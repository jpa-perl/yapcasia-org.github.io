Event Title

大人のYAPC (YAPC for Grown-ups)

Event Description

テーマは「ガチすぎる話」！

普段のYAPCではちょっと発表できない話を事前登録した人のみが聞けるイベントです！

トークの内容は「ガチすぎる話」をしていただきます。ガチでちょっとアレな動画を解析したらわかった様々な事や、ちょっとガチでやばい以前のお仕事など、普通のYAPCセッションで聞ける内容とは趣が異なったミニイベントになる予定です。

  • トーク募集・参加登録は後ほど行う予定です
