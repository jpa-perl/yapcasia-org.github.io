無限コーヒーからあの人が出演するセッションまで... イベントトラックの全貌！

Tweet

おはようございます。会期が近づいてきて完全にYAPCモードなわだです。今回は今まであまり明らかにしてこなかった「イベントホール」という会場についてと、そこで行なわれる「イベントトラック」についてご紹介いたします。

2014年。僕がYAPC::Asiaをやるにあたって、なるべく参加者が交流出来て、かつ、いい意味で「ラフな」セッションが欲しいと思っていました。その結果、唯一飲食可能なイベントホールの会場を最大限に利用しようと、そこに仕掛けをつくり、イベントトラックというなるべく「エンタメ色」の強いセッションを走らせることにしました。その仕掛けとセッションを見てみましょう。

以下がササッと書いてみたイベントホール内の当日配置図です。前方がセッションスペース、後方ではドリンクなどを配布する交流スペースになっています。

無限コーヒー

無限にコーヒーが提供される「無限コーヒー」ブースを後方に設置いたします。ホットコーヒーのみならず、

  • アイスコーヒー
  • その他ジュース
  • 軽いお菓子

も個数が限られる場合がありますが配布いたします。当然無料です。去年僕がYAPC::NAというアメリカのYAPCに参加して「コーヒー飲めるのいいな〜」って思ってたのをそのまま実現しました。以下の写真はそのYAPC::NAの模様です。


ほぼ無限かき氷

DMM.comラボさんのご協力の元「DMM オリジナル かき氷」もあります。量はたくさんあるので「ほぼ無限」に近いかと思います。コンビニとかに売っている「サクレ レモン」みたいな器に入った形での提供です。もちろん無料です。


お水

イベントホール付近以外にも置かれる可能性ありますが、今年は

  • 株式会社ディー・エヌ・エーさん
  • 株式会社Diverseさん

の2社のご協力の元お水が配布される予定です。


時間限定レッドブル

レッドブル・ジャパンさんのご好意で「Wings TEAM」という「レッドブルを配るお姉さん」が2日目/30日(土)の13時から15時まで登場します。みんな大好きレッドブル！たくさん飲んじゃいましょう。


ランチセッション

1日目・2日目と両日12時から限定80個でスポンサーさんのお話を聴きながら無料で豪華お弁当を食べれるセッションを用意しております。これについては後ほどまたアナウンスするかもです。


次にセッションについてサクッと紹介しましょう。

QuizNow!

1日目(29日) 13時から45分間

ディー・エヌ・エーさんが開発した/しているQuizNowというアプリの裏側とそのアプリを使った参加型クイズセッションが行なわれます。

  • QuizNow! - YAPC::Asia Tokyo 2014


ライブコーディング 2014

1日目(29日) 14時から60分間

実際にPerlを使ったアプリケーションをその場でつくっていき完成させます。そのエディタの様子などをリアルタイムで見て学んだり、ハラハラドキドキする企画です。

  • ライブコーディング 2014 - YAPC::Asia Tokyo 2014


地域.pmミートアップ 2014

2日目(30日) 13時から45分間

各地域の「XXX.pm」が大集合。北はHokkaido.pmから南はYomitan.pmまで7箇所以外の.pmが各地方の「美味いもの自慢」とかします。

  • 地域.pmミートアップ 2014 - YAPC::Asia Tokyo 2014


Perlあるある

2日目(30日) 14時から45分間

アメトーーークのエンジニア版みたいなノリで、著名Perlハッカーの皆さんのあるあるネタを探ります。ゲストに @lestrrat, @takesako, そして @miyagawa 氏を迎えお送りいたします。

  • Perlあるある - YAPC::Asia Tokyo 2014


あやかNow!

2日目(30日) 15時から45分間

Rubyアイドルとして知られる「池澤あやか」さんとディー・エヌ・エーアプリ「QuizNow」とのコラボ企画です。実際にプログラミングを題材としたクイズでの参加型対決もあるかも？

  • あやかNow! - YAPC::Asia Tokyo 2014


ってことで盛りだくさん！もし居場所に困ったら是非「イベントホール」へお越しください！
