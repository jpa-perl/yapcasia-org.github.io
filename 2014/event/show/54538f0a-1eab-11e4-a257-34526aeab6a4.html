Event Description

株式会社DNPデジタルコム様 にスポンサードしていただいているランチセッションです。

豪華お弁当 を 先着80名 まで無料で配布いたしますので、会場のイベントホール内で味わって下さい！その際、DNPデジタルコム様より以下のプレゼンテーションを発表していただきます。

DNPデジタルコムはICT＆BPO事業を行っており、ホスティングから開発、運用までの一貫したソリューションを提供しています。今回はYAPCの場を通して皆さんに当社の開発風景やソリューションの実績などをご紹介させて頂きたいと思います。
