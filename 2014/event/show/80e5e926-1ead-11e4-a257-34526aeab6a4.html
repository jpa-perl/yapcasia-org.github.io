Event Description

Reject conとは?

2014年7月16日. YAPC::Asia 2014に応募された100を越えるトークの中から, 本編で発表の時間が与えられる, 採択されたトークの一覧が発表されました.
採択されたトークは, 約50個. その裏には, 同じ数だけ落選(reject)されてしまったトークがあります:

折角, YAPC::Asiaに向けて練りに練ったトークを, ここで終わらせてしまうのは勿体無い!
ということで, rejectされてしまったトークを披露するイベント, ｢Reject con｣を開催することになりました!

美味しい食事とドリンク, そしてビールと共に, Rejectされてしまったトークを供養しましょう!

※本イベントはYAPC::Asiaの公式イベントではありません. 本イベントに関するお問い合わせは@__papix__までどうぞ!

参加費

今回のReject conは, 最初から食事とドリンクを提供し, 会食しながらトークを披露する/聞くというスタイルで開催します.
そのため, 食事代及びドリンク代として, 入場時に2000円を徴収させて頂きます.
