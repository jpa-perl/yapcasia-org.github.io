タイトル

Stackato - ActiveState's new host-your-own-PaaS

トーク概要

Stackato is the new Private PaaS software from ActiveState. Think of it as a Heroku that you own/host anywhere.

In this talk, Ingy döt Net will show you how to install Stackato, get Perl (and other) applications running instantly, and adapt your own Perl apps to run under Stackato.
